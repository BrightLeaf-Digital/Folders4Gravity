# Getting Started with Folders4Gravity

## Overview

**Folders4Gravity** helps organize your Gravity Forms and GravityView
Views into neatly structured folders in the WordPress admin. This plugin
offers an easy way to create folders, assign forms/views to them, and
keep everything organized for improved management and efficiency.

Once activated, Folders4Gravity adds new options under **Forms → Form
Folders** and **GravityKit → View Folders** (if GravityView is
installed), allowing users to easily group and access forms/views in a
more structured way.

------------------------------------------------------------------------

## Installation

### Step 1: Install the Plugin

1.  In your WordPress admin dashboard, go to **Plugins → Add New**.
2.  Search for “**Folders4Gravity**” using the search bar.
3.  Click **Install Now**, then **Activate** once installation is
    complete.
4.  Alternatively, upload the ZIP file if you have a manually downloaded
    version. Simply go to **Plugins → Add New → Upload Plugin**, select
    the ZIP, and click **Install Now**.

Once activated, the plugin works automatically — no additional
configuration is required.
