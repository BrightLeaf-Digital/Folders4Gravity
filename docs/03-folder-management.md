# Folder Management

Once you have assigned forms and views to folders, you can:

- **Reorder Folders**: Drag and drop folders to reorder them.
- **Rename Folders**: Inside the folder management interface, you can
  rename a folder by typing the new name and clicking **Rename Folder**.
- **Delete Folders**: You can delete a folder if it is **empty**
  (contains no forms/views).
- **Reorder Forms/Views within a Folder**: You can drag and drop
  forms/views inside folders to rearrange their order.

Each folder will display the **number of forms/views** it contains next
to its name for easy reference.
