# Inside a Folder

Within each folder, you will see a list of your assigned forms or views.
You can perform the following actions on each:

- **Edit**: Click to edit the form or view.
- **Entries**: View the entries submitted through the form.
  - **Import Entries**: If **GravityView Importer** is active, you will also see an option to import entries.
- **Settings**: Access form or view settings for quick adjustments.
- **Live Preview / Preview**: Preview how the form or view will look.
  - If **Gravity Perks Live Preview** or **Gravity Forms Dev Tools** (with "Preview Tools" enabled) is active, a **Live Preview** link is shown.
  - If neither is available, a regular Gravity Forms **Preview** link is provided as a fallback.
  - **Note for GF Dev Tools users**: If using Gravity Forms Dev Tools, ensure you have saved the Dev Tools settings at least once with "Preview Tools" enabled (under **Forms → Settings → Dev Tools**) for the Live Preview link to be correctly generated.
- **Create View** (for GravityView users): Create a new view for the
  form. Requires **GravityView** to be installed and active.
- **Copy Shortcode**: Click the shortcode button to copy the form or
  view’s shortcode for use elsewhere on your site.
- **Remove**: Remove the form or view from the folder (without deleting
  the form).
- **Duplicate / Clone**: Create a copy of the form or view within the folder.
  - Forms are labeled **"Duplicate"**.
  - Views are labeled **"Clone"**.
- **Trash**: Move the form/view to the trash (without permanent
  deletion).
