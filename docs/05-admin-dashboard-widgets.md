# Admin Dashboard Widgets

Folders4Gravity adds convenient **admin widgets** on your WordPress
dashboard screen. These widgets provide:

- **Quick links** to your **Form Folders** and **View Folders** (if
  GravityView is installed).
- **Item count** next to each folder to show how many forms or views are
  inside.

These widgets help you quickly access your folders without navigating to
the plugin’s menu.
